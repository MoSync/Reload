Connecting to a Device

1.
Install the Reload Client on your device/emulator.
2.
Launch the Reload Client on the device/emulator.
3.
In the Reload Client, enter the Reload Server IP-address:
4.
In the Reload Client, click Connect. (Client connection details are shown in the botton left panel.)

Starting a Project

1.
Click the button.
2.
Select a template:
  • Web based for a consistent look and feel across all platforms.
  • Native UI based for each platform’s own native look-and-feel.
3.
Name your project (lowercase, alphanumeric characters and underscores only, no spaces).
4.
Click the button.
Your project will be created with the needed HTML, CSS and JavaScript files.

Editing a Project

1.
Highlight the project’s name.
2.
Select Open.
Your project’s files are shown in your file browser.
3.
Open the files you wish to edit in your favourite editor, and save your changes when done. (For example, edit index.html, make some changes, save and Reload to see the result.)

Reloading a Project

1.
Highlight the project’s name.
2.
Click the button.
Your project will be loaded/reloaded on the device/emulator and executed.

Debugging with weinre

1.
Highlight the project’s name.
2.
Click the button.
Your project will be loaded/reloaded on the device/emulator and executed in debug mode.
3.
Go to the Debug tab. Note: If you move to other tabs in Reload, winre disconnects, and you need to select the device in the Targets list again to enable winre.
4.
Use winre tools such as the DOM element inspector and console log to debug your app.

Using the Workbench

1.
Select the Workbench tab.
2.
In the Workbench editor, you can evaluate JavaScript code by
  • typing some JavaScript code
  • selecting the code you wish to run
  • pressing the Do it button at the bottom of the editor view
3.
Output from mosync.rlog and any debug information will appear in the Log tab. If you select a JavScript expression that returns a value, it will be shown in the Log tab (return values of type "undefined" are not displayed).

Using mosync.rlog

1.
To do debug logging, add mosync.rlog statements to your JavaScript code, for example: mosync.rlog("Hello World");
(For further details, read the tutorial.)
2.
Reaload your app by clicking the button.
3.
As your code executes, the mosync.rlog output will be displayed under the Log tab.
