Connecting a Device

1.
Get the Reload Client and install it on your device/emulator (full instructions here).
2.
Start the Reload Client on the device/emulator.
3.
In the Reload Client, enter the Reload Server IP-address:
4.
In the Reload Client, click Connect. (When connected, the client details will be shown below, left.)

Starting a Project

1.
Click ”New Project”.
2.
Select a template:
  • Web based for a consistent look and feel across all platforms.
  • Native UI based for each platform’s own native look-and-feel.
3.
Name your project (lowercase, alphanumeric characters and underscores only, no spaces).
4.
Click Create Project.
Your project will be created with the needed HTML, CSS and JavaScript files.

Editing a Project

1.
Highlight the project’s name.
2.
Select Open.
Your project’s files are shown in your file browser.
3.
Edit the HTML, JavaScript and CSS files, as desired, and save your changes.

Reloading a Project

1.
Highlight the project’s name.
2.
Click the Reload button.
Your project will be loaded/reloaded on the device/emulator and executed.

Debugging Using mosync.rlog

1.
Add rlog statements to your JavaScript code, for example: mosync.rlog("Hello World");
(Tutorial)
2.
Click the Reload button.
3.
As your code executes, the rlog output will be displayed here on the Log tab.
